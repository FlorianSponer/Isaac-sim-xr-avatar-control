# Example Avatar (optional)

`newAva.usd` is a sample rigged avatar (UsdSkel) you can load to try the
`avatar.xr.control` extension without supplying your own character. It uses a
neutral UV-grid placeholder material (`textures/ov_uv_grids_basecolor_1024.png`)
and a standard PBR shader, so it has no external asset dependencies.

This folder is optional — the extension works with any UsdSkel avatar.
